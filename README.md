# simple-vue-app

This is s simple frontend system made on the VueJS Framework (v2.0) along with reqres.in mock API 

## Project setup
```
npm install
```

### Compile and Run (Development Mode)
```
npm run serve

```
npm run build
```

### Lints and fixes files

npm run lint
```
